# ARIMA Time Series Forecasting

Home Assignment 3 — Time Series Forecasting using an ARIMA model.

This repository contains a Jupyter notebook that builds, evaluates, and forecasts with an ARIMA model on Amazon (AMZN) stock price data.

## Contents

- `ARIMA.ipynb` — main notebook with the analysis, modeling, diagnostics, and forecasts.
- `amzn_stock_prices.csv` — historical AMZN price data used by the notebook.

## Requirements

- Python 3.9+
- `numpy`
- `pandas`
- `matplotlib`
- `scipy`
- `scikit-learn`
- `statsmodels`
- `yfinance`
- `jupyter`

Install with:

```bash
pip install -r requirements.txt
```

## Usage

```bash
jupyter notebook ARIMA.ipynb
```
